# 社团外交基础

## 校内社团合作

校内社团之间的外联，通常可以分为两类：

第一类是日常来往（弱关系）

即互相加入对方群聊、日常互动、水群起哄等。

这一类关系高度依赖社团负责人的个人社交关系。例如，同一个寝室或同一个班级中如果有两个社团的负责人，两个社团之间的来往通常会更频繁。但这种联系往往停留在“互相认识”的层面，较难直接转化为深度合作。

第二类是活动联动（强合作）

在举办活动时，与校内其他社团进行联动，往往可以同时吸引双方的受众，从而显著扩大活动的影响力。以思飞为例，思飞通常会与学校天协联合举办观影或科普活动，这类活动既能吸引科幻爱好者，也能覆盖天文爱好者，使参与人群明显扩大。

此外，社团之间的合作往往还能形成分工上的互补。例如，科幻社的一个独特优势在于“讲故事”，而宣传本质上也是一种讲故事的过程。许多社团在宣传上缺乏有效表达方式，而科幻社可以通过选取合适的科幻作品进行内容包装，使活动更具吸引力。比如在无线电相关活动中，可以结合《三体》《我们生活在南京》等作品中的相关情节或主题进行宣传。

同时，联动合作还可以有效降低单个社团的工作压力。例如在讲座类活动中，可以与专职举办讲座的社团合作。由科幻社负责邀请嘉宾，对方社团负责主持和现场流程，从而实现更高效的协作。

大部分学校对社团每年都有考核指标，而多个社团联合举办的活动，在规模和影响力上通常明显优于单一社团（尤其是科幻这类相对小众的社团）。因此，跨社团合作往往可以实现共赢。虽然可能不少社团都有过合作的想法，但由于方向不对口或缺乏交集，往往难以真正落地。

本条目将尝试通过上述提到的技巧，讲解如何基于日常建立的人际关系推动活动联动，以及在原本没有交集的情况下，如何发起跨社团合作。

在实际操作中，与其他社团负责人建立私交，往往能带来一些意想不到的妙处。

以我们学校天协和思飞科幻社举例。

第一点是可以互相宣传活动。

天协的前任社长是我同班同学，现任社长是我直系学弟。如果思飞需要在天协群内发布活动宣传，我只需私下沟通；发布之后，他们通常也会协助@全体成员。

第二点是有益于社团的存活。

思飞最困难的时候，曾一度无法招到足够多的成员，于是挂在天协下方作为一个部，以此维持社团运作，避免社团文化断代。

第三点是有效避免外交事故。

如果思飞成员引发外交问题，天协上门沟通时，就需要我及时出面协调，向我的同学和学弟道歉。很多时候我感觉，我这个社长最大的作用，其实就是负责道歉和当“墙头草”，避免各方真的吵起来。

跨社团举办活动时，如果双方已有私交，往往可以事半功倍；如果没有私交，可以考虑在社团招新时直接前往对应摊位说明来意，或通过中间人获取对方社团群聊或负责人的联系方式。

如果社团之间并不熟悉，在提议合作时需要提供一份清晰的策划案。一些社团对活动审批流程较为严格，只有策划案足够明确，对方才能顺利推进相关流程。策划案通常应包括活动介绍、具体时间安排以及各项任务的时间节点。

这里可以举一个反面案例：思飞曾与学校另一社团（团委下属、专职举办讲座的社团）共同举办讲座。对方策划部对海报制作和宣传的时间节点有严格要求，而思飞平时活动流程较为松散，缺乏细化安排，因此在合作中难以适应对方的节奏。

此外，在合作中应提前划清责任，明确每一项工作由哪个社团完成，并区分主办方（发起者）、承办方（具体执行）和协办方（协助者），以避免后续在工作量认定等问题上产生矛盾。如果双方较为熟悉，也可以共同商讨并完善策划案。

活动结束后可以多多添加对方社团干部的联系方式 ，便于日后来往，也可以通过赠送社团周边或组织聚餐等方式，加强联系。

## 外校社团合作

外校社团合作大体同理，但由于地域限制，大部分社交和共同举办的活动通常以线上形式为主。常见的合作方式包括但不限于活动协办、讲座转播、签名祝福、社刊交换等。

不同学校之间的科幻社团，仅通过线上群聊往往难以建立紧密关系。社团间更深入的交流，很大程度上依赖于线下见面，例如在中国科幻大会、银河科幻大会等活动中，负责人互相添加联系方式，并邀请对方进入各自社团群聊。

如果社团活动需要联动指定学校的社团，但对方联系方式未知，可以请高校幻协群中的科幻世界编辑或河流协助联系。

加入外校社团群，也有助于宣传本校的活动。例如某校举办征文活动时，可以在其他高校群内进行宣传并附上联系方式；如果希望进一步提升活动影响力和对方的宣传积极性，也可以将参与转发的学校列为协办方。此外，在社团群中引入外校科幻爱好者，也有助于提高群聊的活跃度。

# 社团资源获取

## 邀请嘉宾

可以通过多种渠道邀请嘉宾，例如基于各类活动中添加的微信、网络公开联系方式以及个人关系等。一般来说，成功概率从高到低依次为：个人关系，其次是活动中添加的微信，最后是网络公开联系方式。

思飞当下也面临着社团老人毕业后，社交关系清零的问题。在 2023 年银河奖期间，小知添加了大量科幻作家的联系方式，目前大家正在发愁如何继承小知的社交关系。我们目前的解决办法是，在小知毕业前尽量多邀请较为重量级的嘉宾，借活动契机建立新的联系；或者在小知毕业后继续“给思飞打工”，在需要邀请嘉宾时拜托小知帮忙联络。

通常建议优先邀请同一城市的嘉宾，或正好在本地参加活动的嘉宾，因为路费往往难以报销。最优的选择是邀请与活动主题相关的校内老师，这类邀请一般需要辅导老师出面；但如果嘉宾本身对相关方向感兴趣，或有同学与老师相熟，成功率会更高。

如果活动确实需要一位“大人物”来提升整体规格，也可以考虑邀请嘉宾录制视频或通过线上形式参与。例如在“三体日”活动中，让整个活动看起来更为高级，小知联系天瑞说符老师，拜托他在百忙之中录制了一段约一分钟的祝福视频，用于活动宣传。

邀请嘉宾时注意风险 ，不要邀请有犯罪前科、曾发表过不当言论的嘉宾。 同时，活动负责人—定要有政治敏感性，具备基本的风险意识 ，必要时可以终止活动。

如果需要正式的邀请函 ，可能需要由导员出面 ， 以所在学院名义开具邀请函并加盖公章 。具体可以咨询社团负责老师。

## 联系组织

这部分可以参考上文“联系嘉宾办讲座”的相关内容。

与科幻主题较为契合的组织，大多与游戏或动漫相关。部分学校可能会对“电子游戏”“动漫”等话题较为敏感，可以尝试与对方协商，在宣传时使用“科普软件”“艺术创作”等表述进行替代。

# 社团关系维护

## 群聊管理与外交礼节

在线下活动中，邀请外校社团成员进入本校群聊，是一种常见的社交礼节。然而，如果群聊中外校成员过多，很可能会压缩本校爱好者的交流空间。当一群互相熟识的人起哄、玩梗时，“小团体”之外的人往往很难参与其中。思飞的主群也存在类似问题，我认为可以参考以下几种解决方式。

第一，社团负责人可以根据不同主题划分群聊，以主群作为中央联络节点，分出“思飞四十二部”：例如思飞探索编辑部（干饭部，日常分享学校附近的美食并组织聚餐）、美工组（负责社团周边制作）、编辑部（负责社刊制作）、纸上谈兵部（桌游群）等。按主题分类，有助于推进日常工作，同时也能保证本社团同学的参与度。

第二，可以引导外校成员分享科幻资讯，同时也可以将本社团的优质内容搬运到其他社团。但需要注意核实信息来源和内容的准确性，遵守友协群规，避免引战，不要在群内攻击，包括但不限于地域、学校、事件或群友，也不建议在较为敏感的群中讨论涉政涉黄不当话题。

第三，在社团骨干入驻外校社团群聊时，可以同步邀请本社团的吉祥物或官方账号，由其承担日常互动和活动宣传的功能。常见的例子包括思飞的“思宝宝飞”（在加入的群聊中发送每日一句科幻，偶尔宣传思飞的线上活动）以及成中医幻协的“幻幻”（每日科幻小说推荐）。真人互动通常仅限于对方少数骨干，且互动内容大多是起哄女装和男娘，并非所有人都有足够的时间和精力参与水群。而社团吉祥物可以在多个群聊中持续输出内容搬运科幻资讯，提升对方社团群内的科幻浓度。

同时，社团账号也可以由后续成员继承使用，从而避免骨干毕业后，其个人账号所积累的社交关系无法延续。

## 周边交换

在线下活动中，可以携带社团纪念品或周边，与其他社团进行交换。常见的周边包括社团 logo 徽章、社娘钥匙扣、飘带、明信片、贴纸等。

思飞的周边目前分为两类：普通款与礼盒款。普通款主要包括社徽吧唧、社娘钥匙扣以及社团成员摄影明信片，成本可控制在一到两元之间，适用于人数较多的场合或社团间赠礼。礼盒款通常包含立牌、钢笔、帆布包等，一般用于赠送活动嘉宾。

# 案例分析

本条目又称“如何把一个小讲座搞成百人大活动”。以下以剑桥天协相关活动为例进行说明。

活动时间线（12月27日举办活动）：

| 时间      | 任务                                                         | 备注                                                         |
| --------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 11月22日  | 拉群                                                         | 注意活动群与策划群分开，注意软件群聊人数上限                 |
| 11月30日  | 线上会议商量活动内容                                         | 讨论活动形式 ，嘉宾提出讲座内容                              |
| 12月15日  | 根据嘉宾时间安排确定活动时间 ，完成海报制作 ，拉活动群 ，开始宣传 ，邀请到杨晚晴老师为活动录制祝福视频 | 审批同步报给导员 ， 由导员出面完成                           |
| 12月16日  | 完成宣传文案                                                 |                                                              |
| 12月18日  | 根据之前活动获得的邮箱 ，邀请到意大利科幻作家FrancescoVerso录制视频 |                                                              |
| 12月19日  | 嘉宾完成大纲                                                 |                                                              |
| 12月20日  | 发布公众号                                                   |                                                              |
| 12月21 日 | 梨洲撰写活动大纲                                             | 明确活动各部分流程                                           |
| 12月24日  | 嘉宾完成讲座PPT ，开始印刷海报                               | 这个时间点偏晚，审批也没能跑完。海报也是到活动当天才拿到 ，最后也没能张贴 |
| 12月25日  | 宣传视频剪辑完成 ，各社团简介PPT完成                         |                                                              |
| 12月26日  | 活动彩排                                                     | 全流程检查 ，确保视频和 PPT 正常播放                         |
| 12月27日  | 活动当天                                                     |                                                              |
| 后续      | 新闻撰写与发布公众号 ，活动证书礼品发放                      |                                                              |

活动的起因是梨洲去剑桥读研时加入了剑桥天协，大家讨论了很久如何与剑桥天协联合举办活动。最初考虑共同观测同一天文现象，但由于剑桥设备较为老旧，双方又都是光学望远镜，受天气影响较大，且近期没有合适的天文现象，因此放弃，转而决定先尝试举办一场线上讲座。

线上活动一方面有助于扩大影响力，另一方面也避免了教室申请和审批等流程上的麻烦（如果参与人数较少，线上形式也不会显得过于尴尬）。

活动前 12 天开始拉群，最初使用微信群。思飞和天协分别在各自群聊中进行宣传，并承诺各大社团转发后可作为协办方领取证书。没想到反响非常积极，微信群很快达到 200 人上限，需要管理员手动拉人，于是临时转为 QQ 群；不到一天 QQ 群也满员，只好紧急转交给有会员的成员扩容。这也算是一次教训：微信并不适合作为大规模活动群聊工具。

人数规模的迅速扩大的夸张规模把大家都吓住了。为了让活动质量与参与规模相匹配、提升整体体验，我们开始紧急邀请更多嘉宾。

同时，为了提高参与积极性，还增加了有奖问答环节。最先在评论区答对问题的参与者可以获得思飞周边。考虑到寄送周边的运费可能比周边本身还贵，有奖问答的数量控制在六个以内。活动结束后，我们制作了证书模板，并根据转发名单以组织为单位生成了 40 多页 PDF，在活动群内统一发布。

总之，非常感谢合作的各个社团以及各大高校的天文社、科幻社，感谢大家的共同努力，将这场活动扩展到了百人规模。
